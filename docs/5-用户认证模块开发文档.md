# 用户认证模块 - 开发文档

## 1. 模块概述

用户认证模块负责系统的登录、注册、登出功能，支持三个角色（患者、管理员、医生）的独立认证流程。

## 2. 功能列表

| 功能 | 说明 | 对应URL |
|------|------|---------|
| 首页 | 系统入口页面 | GET / |
| 患者登录 | 患者身份验证 | GET/POST /login |
| 患者注册 | 新患者账号注册 | GET/POST /register |
| 管理员登录 | 管理员身份验证 | GET /admin-login + POST /admin/login |
| 医生登录 | 医生身份验证 | GET /doctor-login + POST /doctor/login |
| 登出 | 清除回话 | GET /logout |
| 个人中心 | 查看/修改个人信息 | GET/POST /profile |

## 4. 核心业务逻辑

### 4.1 登录认证

```java
// UserServiceImpl.login()
① 根据username查找用户（findByUsername）
② 校验密码是否匹配
③ 校验角色是否符合登录入口要求（患者→role=0，管理员→role1, 医生→role2）

```

### 4.2 角色登录入口

| 角色 | 登录页面 | 后端校验 | 跳转 |
|------|----------|----------|------|
| 患者 | /login.html | POST /login → role==0 | → 首页/ |
| 管理员 | /admin-login.html | POST /admin/login → role==1 | → /admin/dashboard |
| 医生 | /doctor-login.html | POST /doctor/login → role==2 | → /doctor/dashboard |

### 4.3 注册功能

```java
// HomeController.java - POST /register
① 检查用户名是否已存在
② 检查手机号是否已存在
③ 密码一致性校验
④ 创建User对象 role=0（患者）
⑤ 保存到数据库
⑥ 自动重定向到登录页
```

### 4.4 登出功能

```java
// HomeController.java - GET /logout
① session.invalidate() 销毁回话
② 根据referer判断来源
③ 患者→重定向/login，管理员→/admin-login，医生→/doctor-login
```

## 5. 会话管理

使用HttpSession管理登录状态，关键数据结构：

```java
session.setAttribute("user", user);  // 用户完整信息
```

未登录保护：
```java
User user = (User) session.getAttribute("user");
if (user == null) {
    return "redirect:/login";
}
```
