# 后台管理模块 - 开发文档

## 1. 模块概述

后台管理模块面向管理员角色，提供科室管理、医生管理、预约管理、用户管理和数据统计等功能，是系统运维管理的核心模块。

## 2. 功能列表

### 2.1 科室管理
| 功能 | 说明 | 对应URL |
|------|------|---------|
| 科室列表 | 分页查看所有科室 | GET /admin/departments |
| 新增科室 | 填写科室信息 | GET /admin/department/add + POST /admin/department/save |
| 编辑科室 | 修改科室名称和描述 | GET /admin/department/edit + POST /admin/department/update |
| 停用科室 | 软删除，同时停用该科室所有医生 | GET /admin/department/offline所有医生 | GET /admin/department/offline |
| 启用科室 | 恢复科室使用 | GET /admin/department/enable |

### 2.2 医生管理
| 功能 | 说明 | 对应URL |
|------|------|---------|
| 医生列表 | 分页查看所有医生 | GET /admin/doctors |
| 新增医生 | 填写医生信息+创建登录账号 | GET /admin/doctor/add + POST /admin/doctor/save |
| 停用医生 | 软删除医生 | GET /admin/doctor/offline |
| 启用医生 | 恢复医生使用 | GET /admin/doctor/enable |

### 2.3 预约管理
| 功能 | 说明 | 对应URL |
|------|------|---------|
| 预约列表 | 分页查看，多条件筛选 | GET /admin/appointments |
| 更新状态 | 修改预约状态（确认/完成/取消） | POST /admin/appointment/updateStatus |

### 2.4 用户管理
| 功能 | 说明 | 对应URL |
|------|------|---------|
| 用户列表 | 分页查看所有用户 | GET /admin/users |
| 搜索用户 | 按用户名或姓名模糊搜索 | GET /admin/users?keyword=xxx |
| 角色统计 | 统计患者/医生/管理员数量 | GET /admin/users |

### 2.5 数据统计
| 数据统计
| 功能 | 说明 | 对应URL |
|------|------|---------|
| 综合统计 | 概览卡片+图表分析 | GET /admin/statistics |
| 仪表板 | 日常数据概览 | GET /admin/dashboard |

### 4.2 停用科室（级联停用医生）

```java

① 查询科室
② dept.setActive(false)
③ 查询该科室所有医生
④ 逐个 doctor.setActive(false)

```

### 4.3 预约管理多条件查询

```java
// AdminController.java - /admin/appointments
if (有关键字) {
    if (有状态筛选) {
        appointmentService.searchByStatusAndPatientName(status, keyword)
    } else {
        appointmentService.searchByPatientName(keyword)
    }
} else if (有状态筛选) {
    appointmentService.findByStatus(status)
} else {
    appointmentService.findAll() // 全部
}
```

## 5. 角色权限

管理员所有操作前校验：
```java
User user = (User) session.getAttribute("user");
if (user == null || user.getRole() != 1) {
    return "redirect:/admin-login";
}
```

## 7. 图表统计说明

仪表板和统计页面使用 ECharts 5.4.3 CDN 渲染两种图表：
- **预约状态分布饼图**：展示所有预约的状态占比
- **今日预约情况柱状图**：展示当日预约的状态数量
