# 预约挂号模块 - 开发文档

## 1. 模块概述

预约挂号模块是系统的核心功能模块，负责处理患者的预约挂号全流程，包括科室浏览、医生选择、预约创建、预约管理等功能。

## 2. 功能列表

| 功能 | 说明 | 对应URL |
|------|------|---------|
| 科室浏览 | 展示所有可用科室，支持按名称搜索 | GET /departments |
| 医生查看 | 按科室查看医生列表 | GET /doctors |
| 医生详情 | 查看医生详细信息及评价 | GET /doctor/detail |
| 创建预约 | 选择医生和时段创建预约 | GET /appointment/create |
| 提交预约 | 提交预约表单含多重重校验 | POST /appointment/submit |
| 我的预约 | 查看个人预约列表（分页/筛选/搜索） | GET /appointments |
| 编辑预约 | 修改预约信息 | GET /appointment/edit + POST /appointment/update |
| 取消预约 | 取消预约 | GET /appointment/cancel |

## 4. 核心业务逻辑

### 4.1 预约提交流程

```
接收表单数据（doctorId, appointmentDate, appointmentTime, patientName, patientPhone）
    ↓
校验1：预约时间不能是过去时间
    ↓
校验2：预约时间必须在医生坐诊时间内（isWithinAvailableTime
    ↓
校验3：同一医生同一时间段无冲突（hasTimeConflict）
    ↓
校验通过 → 创建Appointment对象 → save() → 返回成功页面
```

### 4.2 时间冲突检测算法

```java
// AppointmentServiceImpl.java
public boolean hasTimeConflict(Long doctorId, LocalDate date, LocalTime time, Long excludeId) {
    List<Appointment> appointments = repository.findByDoctorIdAndAppointmentDate(doctorId, date);
    for (Appointment a : appointments) {
        if (excludeId != null && a.getId().equals(excludeId)) continue;
        int diff = Math.abs(a.getAppointmentTime()... - time...);
        if (diff < 60) return true;  // 间隔<60分钟视为冲突
    }
    return false;
}
```

### 4.3 关键字搜索

患者在"我的预约"页面可按医生姓名搜索：
- `searchByUserIdAndDoctorName(userId, keyword)` - 仅关键字
- `searchByUserIdAndStatusAndDoctorName(userId, status, keyword)` - 关键字+状态组合
